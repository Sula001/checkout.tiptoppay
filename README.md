# TipTopPay Checkout Integration (Python/Flask)

Тестовая интеграция оплаты через криптограмму с поддержкой 3DS.

## Структура
- `/frontend`: HTML форма в темной теме с авто-маской карты.
- `/backend`: Flask сервер для обработки платежей через API TipTopPay.
